# Dcode AGA Lark Base Audit

Project wiki and evidence pack for the May 28, 2026 Dcode AGA Tech Support Lark Base audit.

## Quick Summary

This project documents the first working inventory of Dcode-relevant Lark Bases using the **AGA TECH SUPPORT** account.

Key findings:

- 239 raw Lark search hits were collected.
- 129 visible Base candidates were deduped.
- All 129 candidates were readable through Lark CLI.
- Ownership split: 58 owned by AGA Tech Support, 29 owned by Dcode, 42 shared or other owner.
- `D.136 Class Bible. (admin dcode)` was deep-audited first.
- The D.136 Base contains 26 tables, 2,409 fields, and 157 views.
- Dashboard, workflow, and form details still need extra Lark scopes before the audit is complete.
